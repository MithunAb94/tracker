# Expense Tracker

A full-stack web application that helps users track and manage their daily expenses efficiently.
The application allows users to add, edit, and delete expenses, categorize spending, and view visual insights through an interactive dashboard.

The backend is built with **Java Spring Boot**, providing secure REST APIs and database management, while the frontend is developed using **React** to deliver a responsive and user-friendly interface.

## Features

* User authentication and secure login
* Add, update, and delete expenses
* Categorize expenses (Food, Travel, Bills, etc.)
* Dashboard with spending overview
* Monthly expense tracking
* Data visualization using charts
* Responsive UI for desktop and mobile

## Tech Stack

**Backend**

* Java
* Spring Boot
* Spring Data JPA
* Spring Security (JWT)
* MySQL / PostgreSQL

**Frontend**

* React
* Axios
* React Router
* Chart.js / Recharts

## Architecture

React (Frontend) → REST API → Spring Boot (Backend) → Database

## Goal

This project was built to practice full-stack development using modern technologies and to demonstrate real-world application architecture including REST APIs, authentication, and data visualization.
